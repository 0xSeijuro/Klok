# klok-bot

### 1. Clone Repository

```bash
git clone https://github.com/0xSeijuro/klok.git
```

### 2. Go To Bot Directory
```
cd klok
```

### 3. Install
```
npm install
```

### 4. edit .env
```
nano .env
```

### 5. set up private key
```
nano privateKeys.txt
```

### 6. proxy

```bash
nano proxy.txt
```

### 7. run

```bash
node main.js
```
